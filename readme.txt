# Instalación del Plugin Custom Service en Moodle

Este plugin proporciona un servicio personalizado en Moodle para obtener cursos con funcionalidades de paginación.

## Instalación

1. Copia el directorio `custom_service` en el directorio `local` de tu instalación de Moodle.

2. Accede al panel de administración de Moodle como administrador.

3. Dirígete a "Notificaciones" en el panel de administración para que Moodle detecte el nuevo plugin.

4. Sigue las instrucciones en pantalla para completar la instalación del plugin.

## Observaciones

asegurate de tener un token valido para este servicio

ten en cuenta que debes de tener las opcion "Webservice" y peticiones REST habilitadas
